## What to add

Claude Code now includes Auto-Memory, a feature that enables Claude to learn from its own mistakes over time, improving accuracy and reducing repeated errors across sessions. This persistent memory mechanism tracks what went wrong, why it failed, and how to avoid similar issues in future interactions within the same project.

## Why

Recent Claude Code feature updates (March 2026) introduced Auto-Memory as a core capability that fundamentally changes how Claude Code improves over time. This subsection documents this persistent memory mechanism for users learning Claude Code for the first time, ensuring they understand this learning capability from the start.
