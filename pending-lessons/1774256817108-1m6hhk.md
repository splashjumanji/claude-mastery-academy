## What to add

Claude Code v2.1.69 introduces the /claude-api skill, a built-in capability purpose-built for developers constructing applications on top of the Claude API and Anthropic SDK. Instead of switching between documentation tabs and your editor, you can invoke /claude-api to scaffold API calls, handle streaming responses, and manage tool use patterns directly within your Claude Code session.

## Why

Module 3 teaches Claude Code fundamentals but does not yet cover the /claude-api skill introduced in v2.1.69 (Mar 4, 2026). This new lesson fills that gap and provides a worked example of a key developer feature for API/SDK work within Claude Code.
