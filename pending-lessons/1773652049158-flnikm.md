## What to add

Claude Code automatically enters plan mode on substantial prompts, shifting toward spec-driven development. When you submit a complex or multi-step request, Claude Code pauses to think through the approach before writing code. This structured planning phase ensures careful feature design and reduces rework by aligning on the solution architecture first.

## Why

Claude Code's 2026 auto-plan feature represents a significant shift in how developers interact with AI-assisted coding. This subsection documents the feature and explains its alignment with spec-driven development practices, filling a gap in Module 3's current coverage of the agentic loop.
