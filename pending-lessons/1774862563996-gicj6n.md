## What to add

Claude Code now supports creating interactive artifacts and custom visualizations directly within Claude.ai. You can generate live-rendering React components, data visualizations, and interactive tools that update in real-time as you refine prompts. This transforms Claude from a text-based assistant into a rapid prototyping and visualization platform.

## Why

Claude Code v2.1.76 (March 2026) introduces stable custom visualization and interactive artifact generation capabilities. This feature deserves explicit coverage in Module 2 alongside existing artifact types to reflect the expanded artifact ecosystem and empower users to leverage interactive prototyping.
