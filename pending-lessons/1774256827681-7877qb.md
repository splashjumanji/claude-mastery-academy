## What to add

Prompt caching allows you to cache large context blocks (system prompts, documents, code files) and reuse them across multiple requests, reducing latency and costs. This is critical for production agents that process high volumes of requests with stable, repetitive context.

## Why

Module 4 currently covers context management strategies but does not address prompt caching, which became GA on Feb 4, 2026 and is essential for production agentic workflows. This addition aligns with the module's theme of context optimization and provides engineers with practical patterns for cost and latency reduction.
