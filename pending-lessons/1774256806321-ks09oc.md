## What to add

Remote Control Workflows

Clause Code agents can now be supervised and remotely controlled from mobile and desktop applications. Use the `/remote-control` command in Claude Code to spawn a supervisory session that accepts real-time directives from external devices. This enables mobile devices to monitor, interrupt, and redirect agent behavior without direct terminal access. Perfect for managing long-running agents across distributed systems or supervising agents deployed in production environments. Supervision sessions include live logs, permission checkpoints, and rollback capabilities.

## Why

Claude Code v2.1.69 introduces enhanced hook events with agent_id and agent_type for better agent management and HTTP hooks for web service integration, enabling remote supervision workflows. This new capability should be documented in Module 3 as a core feature of Claude Code sessions.
